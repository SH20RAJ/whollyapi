### Local development

#### Prerequisites

* NodeJS v16+

#### #Run locally

*   Clone the Repo

    ```bash
    git clone https://github.com/sumitkolhe/jiosaavn-api
    cd jiosaavn-api
    ```
*   Install the dependencies

    ```bash
    yarn install
    ```
*   Start development server

    ```bash
    yarn dev
    ```

### Host your own instance
